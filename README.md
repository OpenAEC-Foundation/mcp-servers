# MCP Servers

Model Context Protocol servers connecting AI to AEC tools.

## Revit MCP Server

Python-based MCP server connecting Claude AI to Autodesk Revit. Enables LLMs to query and modify BIM models directly.

Provides 34 tools across categories: status, views, families, selection, parameters, modifications, visualization, IFC queries, and code execution.

## Tech Stack

Python, FastMCP
